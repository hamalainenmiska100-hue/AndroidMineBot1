# AndroidMineBot

Full Android source for MineBot / AFKBot.

## Build
- JDK 17
- Android Gradle Plugin 8.13.2
- Kotlin 2.3.10
- Compose Compiler Gradle plugin enabled

## Backend
The app points to `https://afkbotb.fly.dev` through `BuildConfig.BACKEND_URL`.

## GitHub Actions
A workflow is included at `.github/workflows/android-build.yml`.
